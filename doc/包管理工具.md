参考模板：

[coderwhy](https://www.npmjs.com/package/coderwhy)

## 1、npm

### 1）初始化 package.json

```bash
npm init
# 或直接生成
npm init -y
```

### 2）切换 npm 镜像

```bash
# 切换到官方源
npm config set registry https://registry.npmjs.org/

# 切换到淘宝 npm 镜像
npm config set registry https://registry.npmmirror.com/

# 查看当前镜像
npm config get registry
```
