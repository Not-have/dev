# 常用指令记录

注：npm 支持缓存策略。

## 1、npm 官网

```url
https://www.npmjs.com/
```

## 2、给 package.json 中添加内容

```bash
# 给 scripts 中添加一个 prepare
pnpm pkg set scripts.prepare="husky install"

# 执行
pnpm run prepare
```

## 3、强制重新 build

```bash
npm rebuild
```

## 4、清除缓存

```bash
npm cache clean
```

## 5、本地登录

```bash
npm login
```

## 6、发布

```bash
pnpm publish
```

## 7、让发布的包过期

```bash
npm deprecate
```

## 8、删除

```bash
pnpm unpublish 包名 --force
```
